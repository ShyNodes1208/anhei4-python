# 模板图说明

图像识别用的模板，**必须在 2560×1440 分辨率下截取**。

## 当前模板

| 文件名 | 内容 | 用途 |
|---|---|---|
| `accept.png` | “接受”按钮 | 点重塑后等它出现再点 |
| `not_applicable.png` | 红字“该物品不适用于此配方组合” | 判断该件是否跳过 |
| `recipe_add_affix.png` | 配方“添加词缀”行（图标+文字） | 动态找图点击 |
| `recipe_upgrade.png` | 配方“升级至传奇”行 | 动态找图点击 |
| `recipe_transmute.png` | 配方“嬗变物品”行 | 动态找图点击 |

3 个 `recipe_*` 是从 `start.png`（空盒正常状态）裁出的。配方在放入不同装备后会重排，
脚本用“找图再点”在配方列区域里找到它们的实时位置，找不到才退回固定坐标。

## 参考整屏图（可保留，便于重新裁模板）

- `start.png`：魔盒未动（空盒）时的整屏
- `chuanqi.png`：选中“升级至传奇”后的整屏
- `shanbian.png`：放入武器选中“嬗变物品”后的整屏

## 重新生成

```bash
python d4_capture.py        # 截全屏
python d4_capture.py crop   # 框选裁剪
```
